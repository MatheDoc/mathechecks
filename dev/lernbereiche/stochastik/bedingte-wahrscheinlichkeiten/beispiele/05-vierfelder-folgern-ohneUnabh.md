Wir betrachten die Ereignisse

- $A$: Eine Person putzt sich regelmäßig die Zähne.
- $B$: Eine Person hat gesunde Zähne.

und die zugehörige Vier-Felder-Tafel

|                | $B$      | $\overline{B}$ | $\Sigma$ |
| -------------- | -------- | -------------- | -------- |
| $A$            | $0{,}08$ | $0{,}52$       | $0{,}6$  |
| $\overline{A}$ | $0{,}12$  | $0{,}28$       | $0{,}4$  |
| $\Sigma$       | $0{,}2$  | $0{,}8$        | $1$      |

Dann gilt:

- In der letzten Zeile und der letzten Spalte stehen die Wahrscheinlichkeiten der Einzelereignisse. So entspricht z.B. $0{,}6$ der Wahrscheinlichkeit von $A$, das heißt der Wahrscheinlichkeit, dass eine Person sich regelmäßig die Zähne putzt.
- "$\cap$-Ereignisse": Diese stehen im Inneren der Tafel. So entspricht z.B. $0{,}08$ der Wahrscheinlichkeit von $A\cap B$, das heißt der Wahrscheinlichkeit, dass eine Person sich regelmäßig die Zähne putzt und gesunde Zähne hat.
- "$\cup$"-Ereignisse: Wegen $A\cup B=\\{AB, A\overline{B}, \overline{A}B\\}$ etc. gehören zu jedem "$\cup$-Ereignis" immer genau drei innere Felder. So entspricht z.B. $P(A\cup B)=0{,}08+0{,}52+0{,}12=0{,}72$ der Wahrscheinlichkeit, dass eine Person sich regelmäßig die Zähne putzt oder gesunde Zähne hat (oder beides).
