Eine Vier-Felder-Tafel mit stochastisch unabhängigen Ereignissen $A$ und $B$ habe die Gestalt

|                | $B$     | $\overline{B}$ | $\Sigma$ |
| -------------- | ------- | -------------- | -------- |
| $A$            | $x$     |                | $0{,}7$  |
| $\overline{A}$ |         |                |          |
| $\Sigma$       | $0{,}2$ |                | $1$      |

Da $A$ und $B$ stochastisch unabhängig sind, ist die Vier-Felder-Tafel mutiplikativ, und es gilt $P(A\cap B)=P(A)\cdot P(B)$. Die Wahrscheinlichkeit $x$ ist also $0{,}2\cdot 0{,}7=0{,}14$. Die restliche Vier-Felder-Tafel kann nun wie zuvor weiter vervollständigt werden.
