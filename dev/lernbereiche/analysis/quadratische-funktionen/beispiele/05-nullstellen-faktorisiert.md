Bestimme die Nullstellen von $f(x) = 3(x - 5)(x + 1)$.

- Satz vom Nullprodukt

    Das Produkt $a(x - x_1)(x - x_2)$ wird null, wenn einer der Faktoren null wird:

    $$\begin{aligned}
    x - 5 &= 0 \quad\Rightarrow\quad x_1 = 5 \\
    x + 1 &= 0 \quad\Rightarrow\quad x_2 = -1
    \end{aligned}$$

- Ergebnis

    Die Nullstellen sind $x_1 = 5$ und $x_2 = -1$.

    Der Faktor $a = 3$ beeinflusst die Nullstellen nicht — er bestimmt nur die Öffnungsweite und Richtung.
