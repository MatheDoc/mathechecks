Bestimme die Gleichung der quadratischen Funktion, deren Graph durch die Punkte $A(-1 \mid 6)$, $B(1 \mid 2)$ und $C(3 \mid 6)$ verläuft.

- Gleichungssystem aufstellen

    Ansatz: $f(x) = ax^2 + bx + c$. Die drei Punkte einsetzen:

    $$\begin{aligned}
    f(-1) &= a - b + c = 6 \quad &\text{(I)} \\
    f(1) &= a + b + c = 2 \quad &\text{(II)} \\
    f(3) &= 9a + 3b + c = 6 \quad &\text{(III)}
    \end{aligned}$$

- Gleichungssystem lösen

    (I) + (II): $\quad 2a + 2c = 8 \quad\Rightarrow\quad a + c = 4$

    (I) − (II): $\quad -2b = 4 \quad\Rightarrow\quad b = -2$

    $b = -2$ in (II): $\quad a - 2 + c = 2 \quad\Rightarrow\quad a + c = 4$ (bestätigt)

    $b = -2$ in (III): $\quad 9a - 6 + c = 6 \quad\Rightarrow\quad 9a + c = 12$

    Aus $a + c = 4$ folgt $c = 4 - a$. Einsetzen:

    $$9a + (4 - a) = 12 \quad\Rightarrow\quad 8a = 8 \quad\Rightarrow\quad a = 1$$

    Damit $c = 4 - 1 = 3$.

- Ergebnis

    $$f(x) = x^2 - 2x + 3$$

- Kontrolle

    $f(-1) = 1 + 2 + 3 = 6\;\checkmark$, $\;f(1) = 1 - 2 + 3 = 2\;\checkmark$, $\;f(3) = 9 - 6 + 3 = 6\;\checkmark$
