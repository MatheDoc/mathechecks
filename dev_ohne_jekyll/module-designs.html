3. Extremstellen

Eine Funktion f besitzt an der Stelle x₀ eine Extremstelle, wenn f dort ein lokales Maximum oder Minimum annimmt. Um Extremstellen zu finden, untersucht man die erste Ableitung f′.

Definition
Notwendige Bedingung: Wenn f an der Stelle x₀ ein lokales Extremum hat und f an x₀ differenzierbar ist, dann gilt f′(x₀) = 0.
Hinreichende Bedingung (Vorzeichenkriterium)
Wenn f′(x₀) = 0 und f′ bei x₀ das Vorzeichen wechselt, dann hat f an x₀ ein Extremum:
+ → −: Hochpunkt  |  − → +: Tiefpunkt  |  kein Wechsel: Sattelpunkt

Alternativ kann man das zweite Ableitungskriterium nutzen: Wenn f′(x₀) = 0 und f″(x₀) < 0, liegt ein Hochpunkt vor; bei f″(x₀) > 0 ein Tiefpunkt. Bei f″(x₀) = 0 ist keine Aussage möglich.

Beispiel: f(x) = x³ − 3x² + 2 Lernbeispiel
1
Ableitung berechnen
f′(x) = 3x² − 6x = 3x(x − 2)
2
Notwendige Bedingung: f′(x) = 0
3x(x − 2) = 0  →  x₁ = 0, x₂ = 2
3
Vorzeichentabelle von f′
x < 0: f′ > 0 | 0 < x < 2: f′ < 0 | x > 2: f′ > 0
4
Schluss: Extrempunkte
HP: (0 | f(0)) = (0 | 2)   TP: (2 | f(2)) = (2 | −2)
💡
